# 🛡️ SysGuard

**SysGuard** is a modern desktop system diagnostic tool designed to analyze your computer, detect issues, and provide real-time insights into system health.

Built with performance and simplicity in mind, SysGuard helps users understand what’s happening inside their system — without complexity.

---

## 🚀 Features

* 🔍 **System Scan**
  Detect system file issues using built-in OS tools

* 📊 **Real-Time Monitoring**
  Track CPU and RAM usage instantly

* 🧹 **Temporary File Cleanup**
  Remove unnecessary files to improve performance

* ⚡ **Fast & Lightweight**
  Minimal resource usage with maximum efficiency

* 🖥️ **Modern UI**
  Clean and user-friendly interface

---

## 🧠 Tech Stack

* **Desktop Framework:** Electron
* **Backend:** Node.js
* **Frontend:** HTML, CSS, JavaScript

## ⚙️ Installation

```bash
git clone https://github.com/beratcomert/SysGuard.git
cd sysguard
npm install
npm start
```

---

## 🔐 Requirements

* Windows OS
* Administrator privileges (required for system scanning)

---

## 🧪 Usage

1. Launch the application
2. Click **"Scan System"**
3. View system health status
4. Clean temporary files if needed
